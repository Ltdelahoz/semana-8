El comando "uname -a" muestra información sobre el sistema operativo, el núcleo (kernel) y la arquitectura del equipo.

El comando "whoami" muestra el nombre del usuario que está utilizando el sistema.

Conocer esta información es útil porque permite identificar el entorno donde funciona un servidor, verificar el usuario con el que se ejecutan procesos y facilitar tareas de administración y diagnóstico.
